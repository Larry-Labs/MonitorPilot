# Proposal: DDC 扩展控制 & 多显示器管理（v0.2.0）

## Why

v0.1.0 实现了核心的输入源切换功能。用户对显示器的日常控制需求不仅限于输入源——亮度、对比度等参数调节同样高频。同时，多显示器场景下缺乏批量操作和预设管理，影响效率。

## What Changes

### 新增能力

1. **DDC 扩展控制（ddc-controls）**
   - 亮度调节（VCP 0x10）：滑块 0-100
   - 对比度调节（VCP 0x12）：滑块 0-100
   - 音量调节（VCP 0x62）：滑块 0-100（部分显示器支持）
   - 电源模式（VCP 0xD6）：开启 / 待机

2. **多显示器管理（multi-monitor）**
   - 显示器排序：用户自定义 UI 中的显示顺序
   - 输入预设：保存/加载多显示器输入配置组合
   - 批量切换：一键应用预设，同时切换多台显示器
   - 托盘预设快速切换：在系统托盘菜单中直接选择预设

### 修改能力

- **monitor-detection**：扩展 `MonitorInfo` 结构，增加亮度/对比度/音量等当前值
- **system-tray**：托盘菜单新增预设子菜单

## Impact

- 后端：新增 DDC 读写命令（VCP 0x10/0x12/0x62/0xD6）
- 前端：新增控制面板 UI（滑块、预设管理页）
- 配置：扩展 config.json 支持预设和显示器排序
- 跨平台：所有平台均通过现有 DDC 抽象层实现
